# GameVault Blog

A simple blog application built with Express.js and EJS templates, featuring a gaming theme.

## Features

- Blog listing page with featured posts
- Individual blog post detail pages
- Responsive design with Bootstrap
- Dark theme optimized for gaming content

## Deployment

This project is configured for deployment on Vercel.

### Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Or start the production server:
   ```bash
   npm start
   ```

The server will run on `http://localhost:5000` by default.

### Vercel Deployment

1. Push this code to a GitHub repository
2. Connect your GitHub repo to Vercel
3. Vercel will automatically detect the Node.js app and deploy it

The app uses `process.env.PORT` for dynamic port assignment on Vercel.
